📝 Sobre o Projeto: TCC SUS
Sistema de Gestão para o Sistema Único de Saúde (SUS)

🎯 Objetivo
Desenvolver uma ferramenta para melhorar a administração de unidades de saúde, agilizar o agendamento de consultas e gerar relatórios estratégicos para gestores públicos.

🛠 Tecnologias Utilizadas
📡 Backend
Node.js (JavaScript no servidor)

Express (API REST)

Sequelize (ORM para MySQL)

JWT (Autenticação segura)

💻 Frontend
React.js (Interface moderna)

Vite (Build rápido)

Axios (Conexão com a API)

Chart.js (Gráficos para relatórios)

🗃 Banco de Dados
MySQL (Armazenamento de dados estruturados)

✨ Funcionalidades Principais
✅ Cadastro de Pacientes

Dados pessoais, cartão SUS, histórico médico.

✅ Gestão de Unidades de Saúde

UBS, hospitais, UPAs com horários e especialidades.

✅ Agendamento Inteligente

Marcação de consultas com confirmação automática.

✅ Painel Administrativo

Relatórios em tempo real (fila de espera, taxa de ocupação).

✅ Acesso Seguro

Autenticação por token (JWT) para médicos e gestores.

📌 Contexto do TCC
Problema Identificado
Falta de sistemas integrados para gestão do SUS em municípios pequenos, leading to:

Filas longas

Dados desorganizados

Dificuldade na alocação de recursos

Solução Proposta
Um software web que:
✔ Centraliza informações
✔ Automatiza agendamentos
✔ Gera insights para tomada de decisão
